# Sigil Engineering Workflow — Help

Available skills:

- **plan-execute**: Verified, risk-assessed implementation plans with language detection and parallel agent execution
  "Plan this feature implementation" or "Plan this with security review"

- **test-doctor**: Language-agnostic test fixing with root-cause analysis and regression snapshots
  "Fix the failing tests" or "Why are the tests failing?"

- **safe-commit**: Security-gated git workflow with secret scanning, retry logic, and auto-generated messages
  "Commit these changes" or "Push my changes safely"

- **review-apply**: Feedback implementation with TDD for bugs and cross-file dependency tracing
  "Apply this PR review feedback" or "Implement the code review suggestions"

- **solution-ensemble**: Multi-agent solution generation with security evaluation
  "Generate multiple approaches" or "Find the best solution for this problem"
