# Omniscient Parliament

**Multi-perspective AI review engine for Claude Code.**

Spawns parallel expert agents (Engineering, Security, UX, CEO/Strategy, QA) to audit any codebase, design doc, or skill — then produces a consensus verdict with scores and actionable findings.

The quality gate for AI skills marketplaces.

## Usage

```
/parliament review this code
/parliament audit ~/skill/SKILL.md
```
